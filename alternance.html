MetaHuman · gRPC · UE5

A MetaHuman config & dataset-gen tool.

The app communicates via gRPC and/or a UI to the json config file to receive configuration data from an external service, then uses those parameters to drive the MetaHuman and generate datasets — used internally by the team.
