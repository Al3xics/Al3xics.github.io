/ 01 — WHY

"Unreal does it" wasn't enough.

I love Unreal. I live in Unreal. But I wanted to understand what it's actually doing when it renders a frame. So I wrote my own renderer, from scratch, in Vulkan — a modern low-level graphics API.

The scope is deliberately narrow: a working 3D pipeline, a camera I can move around, a transform system, and the ability to load and run shaders.

/ 02 — ARCHITECTURE

A peek at the surface.

// main.cpp — minimal frame loop
auto gfx = VulkanContext::create(window);
auto pipeline = Pipeline::fromShaders(
    "shaders/default.vert.spv",
    "shaders/default.frag.spv"
);

Camera camera;
camera.setPosition({0, 0, 5});

while (!window.shouldClose()) {
    camera.update(deltaTime);         // 6-axis controls
    transforms.tick(deltaTime);       // transform components
    gfx.drawFrame(pipeline, camera, scene);
}
